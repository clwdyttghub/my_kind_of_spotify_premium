# Spotify Premium - Libu-Libong Buwan

A Spotify-themed web music player dedicated to the song "Libu-Libong Buwan" by Kyle Raphael, featuring an interactive 3D photo carousel and real-time synchronized lyrics. Built using HTML, CSS, and Vanilla JavaScript.

## 🚀 Live Demo
You can view and play the live project here: [https://clwdyttghub.github.io/my_kind_of_spotify_premium/](https://clwdyttghub.github.io/my_kind_of_spotify_premium/)

## 🛠️ Built With
* **HTML5** - For structure, modern font icons, and audio integration.
* **CSS3** - For the dark Spotify user interface, dynamic progress bar, and 3D image carousel transforms.
* **Vanilla JavaScript** - For interactive player controls, word-by-word lyric sync, and fluid carousel rotation logic.

## 📂 Project Structure
* `index.html` - The main entry point containing the interface markup.
* `style.css` - Custom layout stylesheets for the layout and responsive design.
* `script.js` - Audio tracking systems, interactive buttons, and slide movement scripts.
